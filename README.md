# senja-touch

A ClojureScript convenience library for Sencha Touch 2.1

## Installation

Add the following to your `project.clj` dependencies:

```clojure
[org.clojars.franks42/senja-touch "0.1.0"]
```

## Usage

On the cljs-side, require sencha.ext library:

```clojure
(ns myapp
  (:require [sencha.ext]))
```
